# A digital clock project for my finals
## Features
- Tells the time
- Has an alarm. (only can take one time)
- Shows the date

## Parts
- As MCU atmeaga328p is used.
- As RTC ds1307 is used.
- A 4 digit 7 segment display is used a output.
